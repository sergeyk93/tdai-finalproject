1) Add to the build path of the project all of the external jars in the lib directory
	1.1) Right click on the project and then hit properties
	2.2) Choose Java Build path 
	1.3) In the libraries tab hit the Add External JARs button
	1.4) Choose the jars from lib directory and hit ok
	
2) In the same page choose NativeFmodEx.jar and uncollapse it
3) Choose Native library location 
4) Choose the location path to be lib/fmod
5) Play!
